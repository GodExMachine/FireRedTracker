FireRed Tracker GUI

This is a tracker for Pokémon FireRed that displays your party Pokémon, colored IVs, and moves.

How to use:

1. Place the FireRedTrackerGUI folder in the same directory as your game ROM.  
   Example:

\Games\roms\
│
├─ FireRedTrackerGUI\

├─ Pokemon - Fire Red.sav

├─ Pokemon - Fire Red.gba

2. Make sure the save and room files are named exactly:

Pokemon - Fire Red.sav

Pokemon - Emerald.sav

Pokemon - Ruby.sav

Pokemon - Saphire.sav
